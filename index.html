HTML & CSS Course

Master the basics of web development with HTML and CSS.

Course Overview

This HTML & CSS course is designed to help you build a strong foundation in web development. You'll learn the essentials of HTML for structuring web pages and CSS for styling and layout. By the end of this course, you'll be able to create beautiful and functional web pages.

HTML Basics

1. Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way.

2. HTML Elements and Tags

HTML elements are represented by tags. Tags are enclosed in angle brackets (<>). Most tags have an opening tag and a closing tag. For example, <h1> is an opening tag, and </h1> is a closing tag.

3. Document Structure

A basic HTML document structure includes the <!DOCTYPE>, <html>, <head>, and <body> tags. The <head> section contains meta information, and the <body> section contains the content of the web page.

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

4. Common HTML Elements

Some common HTML elements include headings (<h1> to <h6>), paragraphs (<p>), links (<a>), images (<img>), and lists (<ul>, <ol>, <li>).

Advanced HTML Topics

1. Semantic HTML

Semantic HTML is the use of HTML to reinforce the semantics, or meaning, of the information in web pages. For example, using <article>, <section>, <header>, <footer>, and <nav> tags to structure your content.

2. Forms

HTML forms are used to collect user input. The <form> element is used to create a form. Inside the form, you can use various input elements like <input>, <textarea>, and <button>.

<form>
    <label>Name: <input type="text" name="name"></label>
    <label>Email: <input type="email" name="email"></label>
    <label>Message: <textarea name="message"></textarea></label>
    <button type="submit">Submit</button>
</form>

3. Tables

HTML tables are used to display data in a tabular format. The <table> element is used to create a table, and <tr>, <th>, and <td> elements are used to define rows, headers, and data cells.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>30</td>
    </tr>
</table>

4. Multimedia

HTML supports multimedia elements like images, audio, and video. The <img> tag is used to embed images, <audio> for audio, and <video> for video.

<img src="image.jpg" alt="Description of image">
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>
<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
</video>

Less Commonly Used HTML Tags

1. <abbr>

The <abbr> tag defines an abbreviation or an acronym. You can use the title attribute to provide the full expansion of the abbreviation.

<abbr title="HyperText Markup Language">HTML</abbr>

2. <address>

The <address> tag specifies the contact information for the author/owner of a document or an article. This tag is typically styled with italic text.

<address>
    Written by <a href="mailto:webmaster@example.com">Webmaster</a>.<br>
    Visit us at <a href="https://example.com">Example.com</a>.<br>
    Box 564, Disneyland<br>
    USA
</address>

3. <bdi>

The <bdi> tag isolates a part of text that might be formatted in a different direction from other text outside it. This is useful for user-generated content where the direction of text is unknown.

<p>Total count: <span><bdi>42</bdi></span></p>

4. <bdo>

The <bdo>dir attribute specifies the direction of the text.

<p>This text goes from left to right.</p>
<p><bdo dir="rtl">This text goes from right to left.</bdo></p>

5. <data>

The <data> tag links a visible content of a tag to a machine-readable translation. If the content is a date/time, the browser will recognize it as a date/time, even if the format is non-standard.

<ul>
    <li><data value="21053">Cherry Tomato</data></li>
    <li><data value="21054">Beef Tomato</data></li>
    <li><data value="21055">Snack Tomato</data></li>
</ul>

6. <dfn>

The <dfn> tag specifies the defining instance of a term. The term is usually marked with the <dfn> tag, and the definition follows.

<p>The <dfn>HTML</dfn> language is the standard markup language for creating web pages.</p>

7. <ins>

The <ins> tag defines a text that has been inserted into a document. Browsers will usually strike through the text that has been inserted.

<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>

8. <kbd>

The <kbd> tag defines keyboard input. It is typically styled with a monospace font.

<p>Select <kbd>Ctrl + C</kbd> to copy text (Windows).</p>

9. <mark>

The <mark> tag defines highlighted text. It is often used to highlight text that is relevant in a given context.

<p>Do not forget to buy <mark>milk</mark> today.</p>

10. <ruby>, <rt>, <rp>

The <ruby> tag is used to mark up ruby annotations, which are often used to provide pronunciation guides for East Asian typography. The <rt> tag defines the pronunciation, and the <rp> tag defines what to show in browsers that do not support the <ruby> element.

<ruby>
    漢 <rt>ㄏㄢˋ</rt>
</ruby>

11. <wbr>

The <wbr> tag represents a word break opportunity—a position within text where the browser may optionally break a line, though its default choice would be not to break the line at that position.

<p>This is a verylongwordthatmightneedtobebrokeninto<wbr>smallerpieces</p>

Practical Examples

1. Creating a Simple Web Page

Here's a simple web page using HTML and CSS:

<!DOCTYPE html>
<html>
<head>
    <title>My Simple Web Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f4f4f9;
            margin: 0;
            padding: 0;
        }
        .container {
            max-width: 800px;
            margin: 0 auto;
            padding: 20px;
        }
        h1 {
            color: #333;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>Welcome to My Simple Web Page</h1>
        <p>This is a paragraph of text.</p>
        <a href="https://example.com">Visit Example.com</a>
    </div>
</body>
</html>
